# Groute Landing

Fleet intelligence landing page for Groute.

## Stack

- **Vite + React + TypeScript**
- **TanStack Router** (file-based routing, `src/routes/`)
- **Tailwind CSS** with custom brand tokens
- **shadcn/ui** primitives (Button, Dropdown Menu)
- **i18next** + `react-i18next` with locale files in `src/i18n/locales/`
  - 🇬🇧 English (`en`)
  - 🇧🇦 Bosanski (`bs`)
  - 🇷🇸 Srpski (`sr`)
  - 🇩🇪 Deutsch (`de`)

## Commands

```bash
npm install
npm run dev     # http://localhost:5173
npm run build
npm run preview
```

## Layout

```
src/
├── main.tsx               # bootstraps router + i18n
├── index.css              # Tailwind + design tokens
├── routeTree.gen.ts       # TanStack Router tree (regenerated by plugin)
├── routes/
│   ├── __root.tsx         # Nav + <Outlet /> + Footer
│   └── index.tsx          # Landing page
├── components/
│   ├── ui/                # shadcn primitives
│   └── landing/           # Nav, Hero, Pain, HowItWorks, Features, Fines, Pricing, CTA, Footer, LanguageSwitcher
├── i18n/
│   ├── index.ts
│   └── locales/{en,bs,sr,de}.json
└── hooks/useRevealOnScroll.ts
```

The language dropdown lives in the nav and footer — selection is persisted to `localStorage`.
